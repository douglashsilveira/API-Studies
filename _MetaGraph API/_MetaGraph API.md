# Graph API

|   |   |
|---|---|
|**A versão mais recente é**:|`v24.0`|

A Graph API é a principal forma de os apps lerem e gravarem no gráfico social do Facebook. Todos os nossos SDKs e produtos interagem com a Graph API de algum modo, e nossas outras APIs são extensões dela. Por isso, é crucial entender como a Graph API funciona.

Se você não conhece bem a Graph API, recomendamos começar por estes documentos:

1. [Visão geral](https://developers.facebook.com/docs/graph-api/overview): saiba como a Graph API está estruturada e como ela funciona.
2. [Introdução](https://developers.facebook.com/docs/graph-api/get-started): explore os pontos de extremidade da Graph API com a ferramenta Explorador da Graph API e faça sua primeira solicitação.
3. [Solicitações em lote](https://developers.facebook.com/docs/graph-api/batch-requests): saiba como enviar várias solicitações de API em uma única chamada.
4. [Solicitações de depuração](https://developers.facebook.com/docs/graph-api/guides/debugging): saiba como depurar solicitações de API.
5. [Como solucionar erros](https://developers.facebook.com/docs/graph-api/guides/error-handling): saiba como lidar com erros comuns encontrados ao usar a Graph API.
6. [Field Expansion](https://developers.facebook.com/docs/graph-api/guides/field-expansion): saiba como limitar o número de objetos retornados em uma solicitação e realizar solicitações aninhadas.
7. [Secure Graph API Calls](https://developers.facebook.com/docs/graph-api/guides/secure-requests): saiba como fazer solicitações seguras à Graph API.
8. [API de Carregamento Retomável](https://developers.facebook.com/docs/graph-api/guides/upload): saiba como carregar arquivos na Graph API.
